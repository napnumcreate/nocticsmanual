一覧の項目には期間内の以下の項目が表示されます。

・売上合計

請求金額の合計が表示されます。

・キャスト支払合計

キャスト支払の合計が表示されます。

・店舗収益合計

店舗収益の合計が表示されます。

※売上合計からキャスト支払いと雑費等を引いて、月極寮費を足したものが店舗収益になります。

・出勤回数

出勤回数が表示されます。

・出勤時刻:合計

出勤時刻の合計が表示されます。

・接客回数

接客回数が表示されます。

・接客時刻:合計

接客時刻の合計が表示されます。

・シフト回数

シフトの回数が表示されます。

・シフト時刻:合計

シフトの時刻の合計が表示されます。

・実行されたシフト回数

出勤をしたシフトの回数が表示されます。

・欠勤回数

欠勤回数が表示されます。
