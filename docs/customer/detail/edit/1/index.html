顧客詳細の画面には以下の項目が表示されます。

表示された項目を入力し顧客として扱うことができます。

・名前

顧客の名前を入力します。

・電話番号

顧客の電話番号を入力することができます。

・ID

顧客のIDを入力することができます。

・来店回数

顧客の来店回数を入力することができます。

・初回来店日

顧客の初回来店日の日時を入力することができます。

※来店回数が0回の場合はこの項目は表示されません。

・最終来店日

顧客の最終来店日の日時を入力することができます。

※来店回数が0回の場合はこの項目は表示されません。

・出禁

顧客に出禁の欄のチェックボックスにチェックを入れると対象の顧客を出禁として登録することができます。

顧客に出禁の欄のチェックボックスのチェックを外すと対象の顧客の出禁を解除をすることができます。

・メモ

メモを入力することができます。

・更新

入力した値で顧客を更新することができます。

・削除

顧客を削除することができます。
