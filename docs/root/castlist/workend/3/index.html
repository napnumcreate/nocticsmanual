待機料・寮費・雑費が一覧で表示されます。

一覧には以下の項目が表示されます。

・待機料・寮費・雑費名

待機料・寮費・雑費名が表示されます。

・適用

適用(スイッチが青の状態)にすると、適用した行の待機料・寮費・雑費がキャスト支払に反映されます。

・キャスト支払金額

キャスト支払金額が表示されます。

キャスト支払の合計から割引のときは金額の最左端に-(マイナス)が表記されます。割増のときは金額のみ表示されます。

待機料・寮費・雑費名の区分によって以下の通り画面表示時の適用の選択状況が変わります。

・雑費(出勤日天引き)の場合、適用状態で表示されます。

・が寮費(出勤日払い)または寮費(カスタム支払い)の場合,適用していない状態で表示されます。

・雑費(接客天引き)の場合、データ管理で作成した金額×接客数の金額が表示され、適用状態で表示されます。

・待機量(日払い)の場合、接客が0件の場合は適用状態、1件以上の場合は適用していない状態で表示されます。
