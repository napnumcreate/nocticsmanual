各タブのキャストカードには以下の情報が一覧で表示されます。

・出勤時間欄

本日のシフトの開始時間が表示されます。

・退勤予定時間

シフトを選択して出勤した場合、選択したシフトの終了時間が表示されます。

・状態

待機中と表示されている場合、そのキャストは現在出勤中で接客、休憩を実行していない状態です。

接客中と表示されている場合、そのキャストは現在接客を終了していない接客があります。

休憩中と表示されている場合、そのキャストは現在休憩中です。

不在と表示されている場合、そのキャストは未出勤です。

・本日予約件数

本日の予約件数と、未実行の予約件数が「(予約件数) 件 : 残 (未実行の予約件数) 件」と表示されます。

・次回予約時間

直近の予約の開始日時が表示されます。

・次回出勤予定

次回のシフト開始日時が表示されます。

※出勤時間、退勤予定時間、本日予約件数、次回予約時間、次回出勤予定は該当しなければ-と表示されます。
