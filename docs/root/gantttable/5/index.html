シフトが登録されているキャストの該当時間は薄オレンジ色に表示されます。

出勤しているキャストの列は該当時間が薄緑色に表示されます。

シフトを登録しておらず、出勤もしていないキャストの列は白色に表示されます。

休憩が登録されているキャストの該当時間は、休憩・休憩時間が表示された灰色のボックスが表示されます。

予約が登録されているキャストの該当時間は、予約・金額・お客様名・接客時間が表示された青のボックスが表示されます。

接客が登録されているキャストの該当時間には、接客・金額・お客様名・接客時間が表示されたボックスが

接客中の接客の場合は緑、接客が終了している予約の場合は灰色で表示されます。

※短い時間の休憩、予約、接客の場合予約または接客または休憩のみが表示され、カーソルを合わせた際に金額、お客様名(予約、接客の場合)、時間が表示されます。

現在時刻以降にカーソルを合わせると、カーソルから次の予約、接客、休憩までまたは3時間分の色が変わります。
