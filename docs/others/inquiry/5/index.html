・追加機能要望

新しく追加してほしい機能や、既存機能をもっと便利にするための要望がある場合に選択します。

（例：新しい集計項目を増やして欲しい／新しい支払い形態の予約をできるようにしたい。）

・不具合報告

画面が正常に動かない、表示がおかしい、操作できないなど、システムの不具合が疑われる場合に選択します。

（例：保存ボタンを押しても反映されない／エラーが出る／画面が真っ白になる）

・利用方法

操作手順や設定方法など、使い方について質問したい場合に選択します。

（例：予約の追加方法が分からない／キャストのステータス変更方法を知りたい）

・プラン・契約内容について

契約中のプラン、料金、請求、契約内容の確認・変更など、契約に関する問い合わせの場合に選択します。

（例：契約プランを変更したい／請求金額の内訳を確認したい）
