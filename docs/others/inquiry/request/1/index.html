追加機能要望のタブを選択した場合は以下の項目が表示されます。

・重要度(セレクトボックス、初期値は空白)

重要度をクリックすると、以下の項目が表示されます。

以下の項目から重要度を選択できます。

　・業務に必須（無ければ業務に支障がある）

　・重要（業務が大きく改善する）

　・便利（効率が上がる）

　・改善要望（利便性向上）

　・アイデア（改善案）

　・その他・不明

・利用頻度(セレクトボックス、初期値は空白)

利用頻度をクリックすると、以下の項目が表示されます。

以下の項目からどのくらい業務で使うかを選択できます。

　・1日に何度も利用する

　・ほぼ毎日利用する

　・週に数回利用する

　・月に数回利用する

　・今回が初めて

　・その他・不明

　・利用タイミング(セレクトボックス)

利用タイミングをクリックすると、以下の項目が表示されます。

以下の項目からどのタイミングの業務で使うか選択できます。

　・予約時

　・キャスト接客時

　・明細確認時

　・出退勤時

　・休憩時

　・シフト管理時

　・その他・不明
