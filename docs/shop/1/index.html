シフト詳細に以下の項目が表示されます。

・名前

表示しているシフトのキャスト名が表示されます。

・開始時刻

シフトの開始日付と時刻が表示されます。

・終了時刻

シフトの終了日付と時刻が表示されます。

・ステータス

キャストのステータスが表示されます。

・編集

編集をクリックするとシフト詳細のシフトの編集の画面が表示され、シフトを編集することができます。

・欠勤

欠勤をクリックするとシフトを欠勤として扱うことができます。

※欠勤にしたシフトの場合は欠勤は表示されません。

※欠勤にしたシフトはダッシュボードとシフト一覧の色が変わります。

・欠勤取り消し

欠勤取り消しをクリックすると欠勤シフトの欠勤を取消し、シフトとして再度表示されます。

※欠勤シフトではない場合は欠勤取り消しは表示されません。
