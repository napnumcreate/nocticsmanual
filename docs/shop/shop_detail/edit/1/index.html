店舗情報の編集の画面に以下の項目が表示され、編集することができます。

・店舗名

店舗名を入力することができます。

・住所

住所を入力することができます。

・電話番号

電話番号を入力することができます。

・メール

メールを入力することができます。

※こちらのメールアドレスはログイン時のメールアドレスではありません

・営業時刻

営業の開始時刻と終了時刻を入力することができます。

24時刻営業の場合は24時刻営業のチェックボックスにチェックをいれます。

日付をまたぐ場合は、終了時刻の最左部に翌と表示されます。

・定休日

定休日として登録したい曜日にチェックをいれます。

・月極寮費

月極寮費の利用中・未使用を選択します。

・支払方法

支払方法の利用中・未使用を選択します。

・入会金

入会金 の利用中・未使用を選択します。

・ホテル

ホテル の利用中・未使用を選択します。

・交通費

交通費 の利用中・未使用を選択します。

・更新

画面左下に表示される更新をクリックすると更新することができます。
