キャスト一覧のキャストのカードには以下の項目が表示されます。

画面右上に表示されるキャスト追加をクリックするとキャストを追加する画面が表示されます。

また、キャストをクリックするとキャストの詳細の画面が表示されます。

・キャスト名検索

入力した文字が含まれるキャストが表示されます。

・キャスト

キャスト名が表示されます。

・身長

キャストの身長が表示されます。

・3サイズ

キャストの3サイズの項目バスト・ウエスト・ヒップが表示されます。

・年齢

キャストの年齢が表示されます。

・カップ

キャストのカップが表示されます。

・ステータス

キャストのステータスが表示されます。

・キャストタグ

登録したキャストのタグが表示されます。

キャストタグ が3件以上の場合は件数が表示されます。

・指名料

登録したキャストの指名料が表示されます。

指名料が3件以上の場合は件数が表示されます。

・オプション

登録したキャストのオプションが表示されます。

オプションが3件以上の場合は件数が表示されます。
