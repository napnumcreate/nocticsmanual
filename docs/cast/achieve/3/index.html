予約・接客のタブを選択すると、以下の項目が表示され、予約・接客を一覧で表示、または予約・接客を検索して表示することができます。

キャスト実績の画面には以下の項目が一覧で表示されます。

・接客・予約日

接客の年月日が表示されます。

・接客・予約

予約の場合は予約と表示されます。

接客の場合は接客と表示されます。

・対応時刻

対応開始時刻と対応終了時刻が表示されます。

・所要時刻

接客・予約の所要時刻が表示されます。

・備考

取消済予約の場合は取消済予約と表示されます。

実行済予約の場合は実行済予約と表示されます。

予約取消をした予約には取消済予約と表示されます。

上記以外の場合は-と表示されます。

・年/月/日

年/月/日をクリックし、任意の年月日を選択すると、選択した年/月/日の予約・接客のみが表示されます。

・予約・接客(セレクトボックス)

予約・接客(セレクトボックス、初期値はすべて)をクリックすると、すべて・予約・接客を選択できます。

すべてを選択した場合は予約・接客が一覧で表示されます。

予約を選択した場合は予約のみが一覧で表示されます。

接客を選択した場合は接客のみが一覧で表示されます。

・クリア

年/月/日・予約・接客(セレクトボックス)の絞込みをすべて解除できます。

表示された行をクリックすることで、予約詳細または接客詳細が表示されます。
