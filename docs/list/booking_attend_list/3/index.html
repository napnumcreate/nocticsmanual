予約・接客一覧の一覧の上に以下の操作項目が表示されます。

・年/月/日、キャスト名、予約・接客(セレクトボックス)、取消済予約、実行済予約表示にて一覧を絞り込むことができます。

※すべて併用して絞り込むことができます。

・年/月/日

年/月/日をクリックし、任意の年月日を選択すると、選択した年/月/日の予約・接客一覧のみが表示されます。

・キャスト名

キャスト名を入力すると、入力した文字を含むキャストの予約・接客一覧のみが表示されます。

・予約・接客(セレクトボックス)

予約・接客(セレクトボックス、初期値はすべて)をクリックすると、すべて・予約・接客を選択できます。

すべてを選択した場合は予約・接客が一覧で表示されます。

予約を選択した場合は予約のみが一覧で表示されます。

接客を選択した場合は接客のみが一覧で表示されます。

・クリア

年 /月/日・キャスト名・予約・接客(セレクトボックス)の絞込みをすべて解除できます。

・取消済予約・実行済予約表示

取消済予約・実行済予約表示を選択すると、取消済予約・実行済予約が表示がされます。

取消済予約・実行済予約表示を選択を解除すると、取消済予約・実行済予約は表示されません。
